## 📌 Project Overview
This project analyzes advertising campaign performance across Facebook and Instagram using an interactive dashboard. It focuses on tracking key marketing metrics, understanding audience behavior, and optimizing campaign performance to improve return on investment (ROI).

---

## 🎯 Objectives
- Track performance of ad campaigns in real-time  
- Compare effectiveness of Facebook vs Instagram  
- Analyze audience engagement and behavior  
- Optimize budget allocation for better ROI  

---

## 📂 Dataset Information
- <a href = "https://github.com/Abin-2002/PowerBI-MetaAds-Dashboard/tree/main/dataset"> Dataset </a>
The dataset simulates real-world Meta Ads performance data, including campaign, ad, and user interaction details.

### 🔑 Key Features:
- Impressions, Clicks, Shares, Comments  
- Purchases (Conversions)  
- Ad Type (Image, Video, Carousel, Story)  
- Platform (Facebook, Instagram)  
- Audience Demographics (Age, Gender, Country)  
- Campaign Budget  

---

## ⚙️ Methodology
- Data cleaning and preprocessing  
- Handled missing and inconsistent values  
- Created calculated KPIs (CTR, Conversion Rate, Engagement Rate)  
- Converted timestamps into meaningful formats  
- Built interactive dashboard using charts and slicers  

---

---

## 📈 Key Insights
- High engagement but low conversion rate  
- Females aged 18–30 are the most active audience  
- Video and Story ads perform better than other formats  
- Ads perform best during afternoon and evening  
- Certain regions (e.g., India, Brazil) show higher engagement  

---

## 💡 Recommendations
- Improve landing pages and conversion funnel  
- Focus on high-performing ad formats (Video, Stories)  
- Optimize campaigns for target audience (18–30 age group)  
- Allocate more budget to top-performing regions  
- Use retargeting strategies to increase conversions
